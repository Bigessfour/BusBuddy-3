# BusBuddy MCP Tools Configuration

## Database Tools

- **bb-sql-query**: Execute safe SELECT queries against BusBuddy database
- **bb-db-health**: Check database connection and table status
- **bb-schema-info**: Get database schema information

## Build Tools

- **bb-build**: Build BusBuddy solution with configuration options
- **bb-clean**: Clean build artifacts and temporary files
- **bb-restore**: Restore NuGet packages with cache clearing

## Project Management

- **bb-health**: Comprehensive system health check
- **bb-run**: Launch BusBuddy WPF application
- **bb-test**: Run unit tests with coverage reporting

## Syncfusion Tools

- **bb-syncfusion-check**: Verify license and package status
- **bb-xaml-validate**: Validate XAML for Syncfusion-only controls
- **bb-theme-check**: Verify FluentDark/FluentLight theme usage

## Logging and Monitoring

- **bb-logs**: Access recent application logs with filtering
- **bb-debug**: Start debug output monitoring
- **bb-errors**: Show recent error reports

## Azure Integration

- **bb-azure-status**: Check Azure SQL connection
- **bb-deploy-status**: Verify deployment readiness
- **bb-config-check**: Validate Azure configuration

## Git Operations

- **git-status**: Repository status with staging info
- **git-diff**: Show changes with file-specific options
- **git-commit**: Commit with validation checks
- **git-branch**: Branch management and remote tracking
